# VibeUI

**需求即代码、设计即代码**

---

## ✨ 简介

Vibe UI（IDE插件版本）依托 MyBricks.ai社区，将可视化设计平台与Vibe Coding相融合，为开发者/产品经理/设计师提供方便的Vibe UI开发体验。

---

## 🛠️ 主要特性
1. 全新的Vibe UI交互体验 <br/>
2. 多重围栏、可控生成 <br/>
3. 需求即代码、设计即代码 <br/>
4. 可完全定制<br/>

## ⚙️ 使用过程

1. **安装插件**：在 VSCode 插件市场搜索 `VibeUI` 安装。
2. **新建画布**：新建一个 `.ui` 文件，或者右键选择「用 MyBricks 打开」。
3. **唤起 AI**：在设计器内点击 AI 按钮，输入你的需求（Prompt）。
4. **Vibe 一下**：生成UI，然后根据需要微调。
5. **导出使用**：选中页面，点击导出，获取代码。
